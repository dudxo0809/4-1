주요 사건과 인물)

포에니전쟁(한니발 전쟁)

	-> 한니발: 로마가 거대한 제국으로 발전하는데 결정적 공헌을 한 사람
		-> 한니발 vs 스키피오
		-> 한니발과의 전쟁을 겪으면서 로마가 크게 발전함
		-> 한니발의 목적 : 로마의 멸망
	-> 로마는 다른나라보다 못한 부분을 인정하고, 자신이 잘하는 군사력에 바탕해서
		-> 정복한 도시를 로마와의 공동체로 만듬
	-> 한니발은 로마만 점령하지 못하고 로마 주변만 점령함
		-> 그럼에도 불구하고 로마제국의 동맹 시스템은 무너지지 않음

	-> 스키피오 : 한니발의 전략 등을 그대로 배움
		-> 로마가 한니발이 지휘하지 않는 전투에서 이김
	** 위대한 전략전술가 vs 로마의 제도적 힘 **

카이사르의 등장전에도 로마 공화정을 개혁하려함

	-> 번번히 실패
	-> 카이사르는 로마에게 제국으로의 길을 제시함
	-> 카이사르는 아주 유능한 장군이자 정치가

	-> 옥타비아누스를 거쳐 공화정이 점점 개혁되어감


포에니 전쟁의 영향)

	-> 로마군에 혁신적인 전술 전법 전수
	-> 로마와 카르타코 패권전쟁에서의 승리
	-> 로마식 통치체제의 우수성 확인


카이사르)

	-> 민중파 귀족 출신 : 원로원의 기득권에 도전
	-> 갈리아 정복 : 프랑스쪽 땅인 갈리아를 정복함으로서 로마의 안보 확보
		-> 권력의 기초가 됨, 이후 제국통치의 효율성을 제고함
	

아우구스투스)

	-> 카이사르가 후계자로 정하기 전까지 아무것도 없는 10대 소년이었음
	-> 안토니우스와 손을 잡고 원로원세력을 숙청
		-> 이후 안토니우스와의 내전에서 승리함
	-> 카이사르보다 정치적 능력은 뛰어났다
	-> 로마제국의 기틀을 마련함 : 카이사르의 착실한 계승자

	영토가 크게 확장됨
