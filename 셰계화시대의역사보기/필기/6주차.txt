
중화제국의 형성

	-> 진시황의 전국통일 : 중화제국의 기틀 마련
	-> 한 고조의 재통일 : 중화제국 황제의 모델
	
	한족과 이민족 왕조를 비교하고,
	중화제국과 서구 제국을 비교해보자

진시황 : 인프라
한고조 유방 : 항우는 원래 대장군집안사람, 유방은 걍 동네 건달이었음

	-> 항우는 왜 유방에게 졌는가?
	-> 유방 : 인과 예 중시, 약법삼장, 적에대한 관용
	-> 항우 : 적에대한 잔인함, 전공을 적절히 나누지 않는 인색함
	-> 항우는 민심을 잃는다

	한신 : 항우에게 등용된 인재 : 과하지욕
	-> 항우는 한신의 능력을 인정하지 못함
	-> 유방에게 등용, 한 통일후 토사구팽 당함

	사면초가의 고사와 함께 항우는 혼자 남고 자결함

최후의 승리는 중국 최초의 농민 출신 황제인 유방

	

중화제국의 기본 특징

	-> 유교를 정치 철학이자 국가 이념으로 함
	-> 황제 중심의 관료제
	-> 황제가 임명한 과리가 군현제 실시
	-> 과거제도를 통한 인재 선발



칭기스칸

	-> 중국을 최초로 통일한 이민족 출신 정복자
	-> 원나라

원나라는 어떻게 중국을 통치했나

	-> 철저한 민족 차별정치 : 엄격한 신분제

강희제 : 중국황제의 모범을 보인 이민족 황제
	
	-> 중국대륙을 실질적 재통일 : 삼번의 반란 진압, 대만정복
	-> 현재 중국의 영토를 거의 확정
	-> 이후 100년간 이어지는 청나라 태평성대의 기반을 구축

원 이후 청나라

	-> 당근과 채찍
	-> 강경책 : 호복, 변발 등 만주 풍습 강요, 반청사상 탄압
			한족과 만주족 혼인금지
	-> 회유책 : 한인학자 우대, 중국문화 존중, 조세감면,
			중국 전통 통치방식으로 중국을 통치
