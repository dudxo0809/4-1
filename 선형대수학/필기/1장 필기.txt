벡터란 무엇인가??

	-> 사과와 오렌지의 갯수를 더할 수는 없다!
	-> 각 과일의 갯수를 벡터로 표현가능하다!

	-> 덧셈을 한다면 벡터의 각 자리의 값끼리 더해 계산이 가능하다

	-> 벡터에 스칼라 값을 곱할 수 있다.


	-> 앞으로 배울 선형대수학은 
	-> 벡터값을 더하거나, 스칼라 값을 곱하는 식의 연산이 주 가 된다!!

p. 6

내적??

	-> 벡터 v와 벡터 w가 있다고 할때 두 벡터의 내적은
	-> 각 벡터의 component를 곱하고 더한 값이다. // v1*w1+v2*w2

	-> 연필 지우개 볼펜의 가격과 각 물품이 팔린 갯수를 벡터 p와 q라고 해보자
	-> 전체 매출은 어떻게 계산할까??
	-> 두 벡터값의 내적을 계산하면 된다!!

	-> ***** 내적은 두 벡터의 절댓값 * cos세타 값으로도 표현된다 *****
	-> 이게 무슨 의미를 갖고 있는가???
	-> cos세타 값에 의해 내적의 값이 완전히 달라진다!
	-> 벡터가 비슷한 방향이면 값이 커지고, 직각이면 값이 0이고, 아예 다른방향이면 음수가 된다!
	
	-> ***** 내적 값은 두 벡터간의 관계를 나타내 준다!!!!!!! *****
	-> 어떤 두 대상이 벡터로 표현될 수 있다면
	-> 두 대상의 내적값을 통해 두 대상의 유사도를 알아낼 수 있다. (normalize는 해야겠지만...)
	-> 각도가 0일때 상관관계가 0인것, 내적이 음수일때는 상관관계가 있다고 본다

Projection

	-> 길이가 1인 벡터에 벡터 u를 projection한다면
	-> 내적 : |u| * cos세타 값이 될것

p. 8

Zero vector?

	-> 두 벡터를 내적할때 두 벡터의 각도가 90도라면 0벡터가 된다
	-> Perpendicular vector

Unit Vector?

	-> 길이가 1인 벡터

p. 9

Unit vector는 어떻게 구할까?

	-> 벡터의 길이로 벡터를 나누면 됨
	-> Unit vector의 x값은 cos세타, y값은 sin세타로 나타낼 수 있다.

Right angle 

	-> 두 벡터의 각도가 직각일때
	-> perpendicular 하다고 한다.

Matrix : 행렬

	-> 3차원 공간을 나타낼 수 있는 3개의 벡터조합을 생각해보자
	-> 3개의 벡터에 스칼라 곱을 적당히 해서 조합하면 3차원 공간 내에서 다양한 위치를 나타낼 수 있다.

	-> 이 Linear Combination을 행렬로 바꿔서 표현할 수 있다.
