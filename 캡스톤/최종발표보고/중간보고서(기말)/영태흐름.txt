<목차>

1. 본 프로젝트 주제 선정의 이유와 목적 - 프로젝트 개요

2. 아노말리란 무엇인가?
	-> 3가지 분류
	-> 정의는 어떻게 된다
	-> 우리는 collective 아노말리를 찾는것을 목표로 한다

3. 사용할 데이터셋
	-> 다변량 시계열 데이터(multivariate time-series data)
	-> 현재 public data set인 HAI dataset사용중
	-> 추후 dataset을 직접 만들 수 있는 모델을 넘겨받을것
	
	-> 이후 data sequence를 Graph의 형태로 바꾸는 작업을 진행할것
	-> 이후 모델소개에서 다시 설명

4. 찾아본 모델
	-> 이전에 사용했던 모델은 LSTM모델이었음
	-> 이전 모델 잠깐 소개
	-> 각 feature들 간의 관계가 중요한 Graph Network쪽으로 노선을 변경

	-> GNN에 대한 설명진행 (대본 섹션3)
	-> 대본 섹션3의 다양한 GNN모델 설명
	-> 이 중 우리가 사용할 모델은 STGCN이다
	-> STGCN은 시간적 공간적 특성을 모두 고려한다는 특징이 있음

	-> 해당 모델을 이해하는데 참고한 논문 (E-STGCN)잠깐 소개
	-> 해당 모델의 구조
		-> 1. data sequence를 Graph의 형태로 변환
		-> 2. input data에 행렬값 추가 (정점 사이의 정보(유사도, 라플라시안))
		-> 3. 데이터 노말라이즈
		-> 4. 컨볼루션 연산 진행
		-> 5. temporal residual layer에 데이터를 넣어 temporal dependency 계산
		-> 6. 5의 결과를 HPN layer에 넣어 RUL을 계산함
			-> RUL은 시스템의 남은 수명이므로, 이 값이 비정상적으로 작으면 이상이 있는것!
		-> 7. 모델의 파라미터가 수렴할때까지 반복
	
	-> 이 모델을 찾은 이유
		-> 각 feature간의 dependency뿐만 고려하는 전통적인 GNN과는 달리
		-> STGCN은 시간적, 공간적인 dependency까지 고려한 모델이기 때문
		-> 기존의 데이터셋과의 호환을 위해, 논문에서처럼 데이터셋의 shape를 변형해야함
	
5. 추후
	-> 기존 HAI dataset을 구조화된 데이터셋(Graph형태)로 변형시켜
